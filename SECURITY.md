# Security Policy

## Supported Versions

This is a small personal project. Only the latest version (main branch) is actively maintained and receives security updates.

| Version       | Supported          |
|---------------|--------------------|
| Latest (main) | :white_check_mark: |
| Older versions| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Preferred way:** Open a private security advisory on GitHub:

1. Go to the [Security tab](https://github.com/dariush315/online-pdf-workspace/security) of this repository.
2. Click **"Report a vulnerability"** (or use this direct link: https://github.com/dariush315/online-pdf-workspace/security/advisories/new).

### What to include in the report:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested fix (optional but very welcome)

I will try to respond and address valid reports as quickly as possible (usually within a few days).

---

**Note:** This project is a client-side / serverless PDF workspace. Most security concerns will likely relate to the web client (e.g. XSS, malicious PDFs) rather than server-side issues.
